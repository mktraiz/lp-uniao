# Template de marca/escola

Copie esta pasta para criar uma nova marca ou use:

```powershell
npm run new:school -- <slug-da-marca> <slug-da-escola>
```

Cada escola deve manter duas frentes:

- `lp-captacao`: captacao continua ou matriculas.
- `lp-campanha`: campanha especifica, evento, bolsa, visita, simulado ou unidade.
